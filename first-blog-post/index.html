Final Exams over + New Blog powered by Ghost

Finally done with my final exams yesterday, so at long last I have time to update my résumé and dabble with Ghost to build my own blog. (No points for guessing how I got my blog profile pic xD)

I've been contemplating for quite a while about building a blogging engine from scratch just to gauge my own coding proficiency. However doing so would have take up ALOT of my time & while researching for ideas, I stumbled upon Ghost - an open source project which caught my attention.

It kind of met my requirement as I wanted something that allows me to create post easily & I decided to use it after finding a guide to host it on Github pages for FREE. =)

Setting up Ghost was pretty quick, the difficult part was to get my hand-picked customized theme (UNO ZEN) to work cause there were lots of new stuff that I haven't used before (coffeescript, ember.js, gulp etc.). After spending the whole Sunday afternoon performing lots of trial-and-error to tweak the theme, I managed to get the hang out of it & started writing my first post!

Things left that I might want to do:

  • Implement google analytics into my site
  • Create & link my portfolio page
  • Make comments available for my posts (Disqus)
  • Add social media sharing buttons
  • Update favicon
  • Fix search bar + broken links on Github pages =/
